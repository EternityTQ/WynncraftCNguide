# The Maiden Tower(level 51)
**塔中女影**

1.找到*Sherk*`[-2172 54 -5118]`

2.跟随地上的石头痕迹前往塔楼`[-1997 81 -5287]`，然后在塔楼边和*Sherk*对话

3.在**Olux**的地下墓穴`[-1742 66 -5593]`里找到骨灰盒

:::tip
解密的顺序是 紫 黄 红 绿
:::

4.返回塔楼，再次和Sherk对话

5.爬上塔楼顶层，触发剧情

6.杀死女巫

奖励
+ 77500 经验
+ 512 绿宝石

:::tip
在Sherk打碎的玻璃下`[-2000 81 -5300]`，你还可以找到一点关于这个女巫的小剧情
:::
